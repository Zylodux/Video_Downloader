🚀 Setup & Requirements   🚀 Kurulum ve Gereksinimler


1.UI Library: Download ReaLTaiizor (v3.7.9.5 or latest) 

Arayüz Kütüphanesi: ReaLTaiizor (v3.7.9.5 veya güncel) indirin.
Link : https://github.com/Taiizor/ReaLTaiizor/releases/tag/v3.7.9.5

2.FFmpeg: Get "git-full" build from Gyan.dev.

FFmpeg: Gyan.dev üzerinden "git-full" sürümünü indirin.
Link : https://www.gyan.dev/ffmpeg/builds/

🛠 Installation Steps  🛠 Kurulum Adımları


1.Create a new folder on your computer. /
Bilgisayarınızda yeni bir klasör oluşturun.


2.Extract and move ffmpeg.exe and yt-dlp.exe into this folder./
ffmpeg.exe ve yt-dlp.exe dosyalarını bu klasöre kopyalayın.



3.Launch the Video Downloader application/
Video Downloader uygulamasını başlatın.



4.Click Settings and select the folder you created./
Settings (Ayarlar) kısmından oluşturduğunuz klasörü seçin



5.Set your MP4/MP3 output location./
Videoların kaydedileceği MP4/MP3 klasörünü belirleyin.



📖 How to Use 📖 Nasıl Kullanılır?


  
1.Copy a YouTube video link./
Bir YouTube video linkini kopyalayın.



2.Paste it into the URL field./
Uygulamadaki link alanına yapıştırın.



3.Press Start to begin downloading./
İşlemi başlatmak için Start butonuna basın.
